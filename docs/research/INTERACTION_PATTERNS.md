# Interaction Patterns

- Intersection-based reveal: opacity 0 → 1 and translateY 5% → 0.
- Stagger increments: approximately 80ms.
- Menu panel: height expansion over 400ms, then link reveal.
- Button icon: scale-out circle + translate-in arrow over 300–400ms.
- Route paths: SVG dash animation tied to viewport entry/scroll progress.
- Flower orbit: multiple absolute layers with different transform speeds.
- Hero video: autoplay, muted, loop, cover.
