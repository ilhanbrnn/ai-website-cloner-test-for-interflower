# Target Technical Stack

- Next.js application with App Router-style streamed markup.
- Tailwind utility CSS plus extensive custom design tokens.
- Next Image optimization.
- Custom local Bricolage font family.
- Roboto Condensed body font.
- Inline SVG for logo, buttons, icons, and animated route lines.
- JavaScript scroll animation with masked text and SVG stroke progress.
- Responsive video source swap between landscape and portrait files.

## Clone equivalents

- Next.js 16 App Router.
- Tailwind CSS v4.
- Local fonts through `next/font/local`.
- React IntersectionObserver wrapper for reveals.
- CSS/SVG line and icon animation.
